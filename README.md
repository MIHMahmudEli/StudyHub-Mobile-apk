# StudyHub Mobile App

A dedicated Android application for the StudyHub platform, providing a seamless mobile experience for students and educators.

## Features
- **Direct Access**: Instantly connect to [StudyHub](https://studyhubb.great-site.net/) via a streamlined mobile interface.
- **Custom Design**: Features a professional graduation cap launcher icon.
- **Back Navigation**: Built-in support for navigating through the website's history within the app.
- **Fast Loading**: Optimized WebView with JavaScript and DOM storage support.

## How to Install
1.  Download the `StudyHub.apk` file from this repository or the shared link.
2.  Open the APK file on your Android device.
3.  If prompted, allow installation from "Unknown Sources" in your device settings.
4.  Complete the installation and launch **StudyHub** from your home screen!
